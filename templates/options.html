Tarot Card Shuffle Draw











    Tarot Card Shuffle Draw is a free and open-source project that 
    simulates a tarot card reading. It is designed for entertainment 
    purposes only and should not be used as a substitute for professional 
    advice. The project is written in Go and uses the Gin web framework.
